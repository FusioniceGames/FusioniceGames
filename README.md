# Fusionice Games Website

## Overview
This project is a website for Fusionice Games, showcasing the games developed by the team and providing an introduction to the team members. The website features a slider on the homepage for displaying images and a dedicated section to introduce the team.

## Project Structure
```
fusionice-games-website
├── index.html          # Main HTML document for the website
├── css
│   └── styles.css     # Styles for the website
├── js
│   └── main.js        # JavaScript for slider functionality
├── images
│   ├── slider         # Images for the slider
│   └── team           # Images of team members
├── data
│   └── team.json      # JSON data for team members
├── README.md          # Documentation for the project
└── .gitignore         # Files and directories to ignore by Git
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd fusionice-games-website
   ```

2. **Open the Project**
   Open the `index.html` file in your web browser to view the website.

3. **Modify Content**
   - Update the images in the `images/slider` folder for the slider.
   - Add team member images in the `images/team` folder.
   - Edit the `data/team.json` file to update team member information.

4. **Styles and Scripts**
   - Customize the styles in `css/styles.css` to change the appearance of the website.
   - Modify `js/main.js` to adjust the slider functionality as needed.
